Where should this progress dashboard live?

This affects the tech, the update mechanism, and how you'll interact with it daily

A

Inside the LearnAgent app

A dedicated /admin or /dev-status page in the Next.js app. Lives in the codebase, auto-updates as you build phases. Only visible to you.

B

Standalone HTML file

A self-contained progress.html file you open in a browser. No server needed, works offline, manually updated. Simple and portable.

C

SpecKit dashboard page

A dedicated Next.js page that reads specs/main/tasks.md and renders live progress. Powered by the actual task files — truth comes from the spec.
